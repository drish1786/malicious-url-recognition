Malicious URL Recognition

Malicious URL Recognition System detects malicious websites. The Project is under progress. The purpose of this project is to create a malicious URL recognition system due to over-rising issues of identity theft, phishing attacks, malware installations which has affected unsuspecting users on the internet.
